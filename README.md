# Generate WorkLog

Generates a worklog temple for daily use

## Description

Generates a worklog temple for daily use

## Getting Started

### Dependencies

* golang. check `go.mod`

### Installing

* `go build` build the project
* `go install` installs the binary `v-worklog` to `$GOPATH`

### Executing program

* `v-worklog -h` provides helpful information
